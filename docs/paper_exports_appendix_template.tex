\section{Hardware}
\label{sec:appendix:hardware}

All experiments were run on a CPU-only server with:
\begin{itemize}
  \item AMD EPYC 9124 CPU
  \item 64 logical CPUs
  \item approximately 1.5~TB RAM
  \item no GPU acceleration for either training or querying
\end{itemize}

\section{How To Rebuild The Bundle}
\label{sec:appendix:rebuild}

From the repository root:
\begin{verbatim}
bash scripts/download_datasets.sh
bash scripts/run_paper.sh
\end{verbatim}

If all experiments have already completed and only the canonical export bundle
needs to be rebuilt:
\begin{verbatim}
PYTHONPATH=src python -m collect --out-root paper_exports \
  --experiments-root ../experiments/adaptivf --data-root ../data
PYTHONPATH=src python -m tables --out-root paper_exports
PYTHONPATH=src python -m plots --out-root paper_exports
PYTHONPATH=src python -m export_bundle --out-root paper_exports
\end{verbatim}
